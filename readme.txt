I have created Magento 2 module creator which will help to developer to create module in Magento 2.

Please follow below step:

Copy and paste magento2-Module creator directory in to your wamp or xampp server directory.

create module using browser with passing Directory Name and Module Name module will be create on "Magento-2-Module Creator\new".

Copy your new created module from "new\app\code" to your project "app\code" folder.

Enjoy with your created module
